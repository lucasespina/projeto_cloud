Sobre o Projeto

Este projeto foi desenvolvido como parte do curso de Cloud Computing no Insper, demonstrando a implementação de uma arquitetura completa na AWS com foco em alta disponibilidade, escalabilidade e disaster recovery.

Este projeto foi desenvolvido por Lucas Espina e Diogo Diniz, estudantes do curso de Cloud Computing do Insper, apaixonados por arquitetura de sistemas em nuvem e soluções escaláveis. A equipe combina expertise técnica com uma visão inovadora para criar sistemas robustos e eficientes.

A equipe combinou conhecimentos de desenvolvimento full-stack, arquitetura de nuvem, segurança e DevOps para criar uma solução robusta e profissional de gerenciamento de imagens.
